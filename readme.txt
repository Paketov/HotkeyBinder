
~~~~~~~~~~~~~~~~~~~Hotkey Binder~~~~~~~~~~~~~~~~

When you press hot keys, it generates keystrokes of other keys.
Also it can run programs by shortcut keys.
	I use it to increase/decrease the volume.

Installation and use:
	1. For install run \Bin\install.bat
	2. For config change \Bin\HotkeyConfig.txt and run \Bin\set_config.bat . For known key codes use \Bin\KeyCodes.txt
	3. Reboot PC or logoff->login
	4. For uninstall run \Bin\uninstall.bat

Have fun!
